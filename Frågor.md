# 1. Definiera Entity Framework Core och dess roll i databashantering för applikationsutveckling.
Svar: Entity Framework är ett object–relational mapping framework från Microsoft som gör det möjligt att redigera databaser med hjälp av objekt orienterad programmering så som C# istället för att använda SQL databaskod som man använder vanligtvis när man utvecklar och underhåller databaser. 
# 2. Beskriv SQL-frågespråket och dess användning för att interagera med databaser, inklusive datahantering och manipulation.
Svar: SQL står för Structured Query Language, och är ett programspråk som används för att redigera, hantera och manipulera relationsdatabaser. Dessa databaser används för att spara på viktig information såsom lösenord, användarnamn, beställningar, bokade möten och produktinformation. 
# 3. Förklara molnbaserad databashantering och hur databaser i molnet fungerar, inklusive fördelar och utmaningar.
Svar: Inom IT används begreppet cloud och moln för att beskriva servrar som oftast ägs av ett företag. Dessa serverar levererar tjänster åt kunder såsom file hosting, website hosting, E- mail hosting,  database hosting, Virtual Private Server(VPS) hosting och video hosting. Google cloud, Google drive, Google Photos, Apple iCloud, Amazon Web Services, Microsoft Azure och Dropbox är alla bra exempel på cloud tjänster/moln tjänster.  
# 4. Sammanfatta DevSecOps och databashantering: hur DevSecOps-principer integreras i hanteringen av databaser på säkerhet och driftskompatibilitet. 
Svar: DevSecOps står för Development Security Operations. DevSecOps är en praktik som består ut av metoder och verktyg för att automatisera utföringen av olika uppgifter och problemlösningar som underlättar jobbet för systemutvecklare och drifttekniker. I dessa automatiseringsprocesser är säkerhet högst prioriterade, vilket förhindrar till exempel databaser från att läcka känsliga uppgifter.
